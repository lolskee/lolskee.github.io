NFL Pass/Rush Prediction Model

03.11.2016

A regression model which predicts whether an NFL play will be a passing play or a rushing play based on time, down, distance to first down, offensive and defensive score, and the actual teams that are playing offense and defense. Use three letter abbreviations for NFL teams such as ‘ATL’ for Atlanta or ‘NE’ for New England. The model was trained using play by play data from 2002 to 2012 (10 seasons). If you're interested in the machine learning you can read more in a paper written by my colleagues and I here. [edit] The server running the prediction model has been suspended but please feel free to contact me if you're interested in trying it.
